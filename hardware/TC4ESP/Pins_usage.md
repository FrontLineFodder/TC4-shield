- D0(IO16) - OT1, heater control
- D1(IO5)  - I2C, SCL
- D2(IO4)  - I2C, SDA
- D3(IO0)  - unused, pulled up, available with restrictions, see ESP8266 notes
- D4(IO2)  - unused, pulled up, output only
- D5(IO14) - unused, available for OT2 or IO3
- D6(IO12) - ZCD Int
- D7(IO13) - BT Rx
- D8(IO15) - BT Tx
- A0       - ADC input, for buttons pad
